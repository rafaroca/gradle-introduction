What is Gradle?

  • General purpose build system
  • Own DSL based on Groovy
  • „build-by-convention“
  • Plugin system (Java, Scala, Groovy, …)
  • Third party plugins available (checkstyle, code coverage, …)
  • Integrates with Ant and Maven
